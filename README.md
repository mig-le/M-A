💖 Nossa História

“Nossa História” não é só um site.
É um pedacinho da nossa vida transformado em algo que fica guardado pra sempre.

Criei esse projeto como uma forma de eternizar momentos, lembrar dos detalhes que fazem a gente sorrir e transformar memórias em algo que pode ser aberto, instalado e revivido a qualquer hora.

Cada cor, cada imagem, cada linha de código carrega intenção.

🌅 Sobre o Projeto

Essa aplicação foi desenvolvida como uma PWA (Progressive Web App), o que significa que ela pode ser instalada como um aplicativo no celular ou no computador.

Mas mais do que tecnologia, a ideia é simples:

Guardar o que é importante.
Valorizar o que é nosso.
Registrar o que faz o coração acelerar.

✨ O que você encontra aqui

📸 Nossos momentos registrados

🎨 Um visual pensado com carinho

📱 Instalação como aplicativo

🌙 Experiência leve, íntima e pessoal

💾 Estrutura preparada para funcionar offline

Porque algumas histórias não merecem depender de internet pra existir.

🛠 Como funciona

O projeto foi desenvolvido usando:

HTML

CSS

JavaScript

Service Worker

Web App Manifest

Tudo isso pra transformar uma simples página em algo que parece um app de verdade.

💌 Por que eu fiz isso?

Porque algumas pessoas entram na nossa vida e mudam tudo.

E quando algo é especial de verdade, merece mais do que fotos espalhadas na galeria.
Merece um espaço só seu.

Esse projeto é isso:
Um espaço nosso.
